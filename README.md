# Broker Rentals MVP

Mobile-first rental listing tool for brokers, agents, and local property connectors to create a personal listings page and share it on WhatsApp and Facebook groups.

## Current focus

- Rental listings only
- Broker login only
- No renter login
- No sale flow in MVP
- No flatmates flow in MVP

## Product idea

Small brokers post many scattered property updates every day. This MVP gives each broker one clean, shareable rental page with photos, videos, pricing, brokerage, and direct contact actions.

## MVP outcomes

- Broker creates profile and first listing in minutes
- Broker shares one public page instead of many statuses
- Renter filters listings by area, budget, room type, and brokerage
- Renter contacts broker directly by call or WhatsApp
